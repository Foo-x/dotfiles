# README

```md
  1-- 2--------------------------------------------------
- (A) task +project @context #tag ~2023-12-31 {key:value}
      3--
    - [ ] subtask
```

1. Priority
    - `(A)` to `(Z)`
    - optional
2. Description with notes
    - Description
        - no prefix
        - required
    - Project
        - prefix: `+`
        - optional
    - Context
        - prefix: `@`
        - optional
    - Tag
        - prefix: `#`
        - optional
    - Due
        - prefix: `~`
        - optional
    - Key Value
        - `{key:value}`
        - optional
3. Checkbox
    - `[ ]` or `[x]`
    - subtask only
    - required
