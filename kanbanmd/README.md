# README

```md
  1-- 2--------------------------------------------------
- (A) task +project @context #tag ~2023-12-31 {key:value}
      3--
    - [ ] subtask
```

1. Priority
    - `(A)` to `(Z)`
2. Description with notes
    - Description
        - no prefix
        - required
    - Project
        - prefix: `+`
    - Context
        - prefix: `@`
    - Tag
        - prefix: `#`
    - Due
        - prefix: `~`
    - Key Value
        - `{key:value}`
3. Checkbox
    - `[ ]` or `[x]`
    - subtask only
